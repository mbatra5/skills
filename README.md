# Agent Skills by Madhur Batra

Reusable AI agent skills for testing, analytics, and automation. Install with a single command to enhance your coding agent.

## Available Skills

| Skill | Description | Install |
|-------|-------------|---------|
| [datalayer-analytics-playwright](./datalayer-analytics-playwright/) | Automated `window.dataLayer` event validation using Playwright (standard + BDD) | `npx skills add mbatra5/skills --skill datalayer-analytics-playwright` |

## Install

```bash
npx skills add mbatra5/skills --skill <skill-name>
```

Or use SkillKit:
```bash
npx skillkit install mbatra5/skills
```

Works with **44+ agents** including: Cursor, Claude Code, GitHub Copilot, Cline, Windsurf, OpenAI Codex, Gemini CLI, Goose, Roo Code, and any agent supporting the [Agent Skills standard](https://agentskills.io/specification).
